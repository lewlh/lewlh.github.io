MyScaleDB еҲҶеҢәдёҺеүҜжң¬зҡ„дҪҝз”ЁиҜҙжҳҺ

йӣҶзҫӨжҳҜеүҜжң¬е’ҢеҲҶзүҮзҡ„еҹәзЎҖпјҢе®ғе°Ҷ ClickHouse зҡ„жңҚеҠЎжӢ“жү‘з”ұеҚ•дёӘиҠӮзӮ№е»¶дјёеҲ°еӨҡдёӘиҠӮзӮ№пјҢе№¶жҸҗдҫӣзҒөжҙ»зҡ„йӣҶзҫӨй…ҚзҪ®ж–№ејҸпјҢеңЁдёҖдёӘйӣҶзҫӨйҮҢй…ҚзҪ®дёҚеҗҢзҡ„е°ҸйӣҶзҫӨпјҢе°ҸйӣҶзҫӨдёӯеҸҜд»ҘжңүдёҚеҗҢзҡ„иҠӮзӮ№гҖҒеҲҶеҢәд»ҘеҸҠеүҜжң¬ж•°йҮҸгҖӮ

MyScaleDB зҡ„еј•ж“Һ

MyScaleDB еҹәдәҺ ClickHouse ејҖеҸ‘пјҢеӣ жӯӨеҹәжң¬дёҠе…Ёзӣҳз»§жүҝдәҶ ClickHouse еј•ж“ҺгҖӮдёҚеҗҢеј•ж“ҺеҜ№дәҺеҲҶеҢәе’ҢеүҜжң¬жңүдёҚеҗҢзҡ„йҷҗеҲ¶пјҢеӣ жӯӨдәҶи§Јеҗ„дёӘеј•ж“Һзҡ„е·®ејӮпјҢжңүеҠ©дәҺжҲ‘д»¬жӣҙеҘҪең°дҪҝз”Ё MyScaleDBгҖӮд»ҘдёӢжҳҜжҲӘжӯўеҲ°20240523дёҖдәӣеёёи§Ғзҡ„ ClickHouse иЎЁеј•ж“Һпјҡ

  • MergeTree зі»еҲ—еј•ж“Һ:
    • MergeTree: еҹәзЎҖеј•ж“ҺпјҢж”ҜжҢҒдё»й”®зҙўеј•гҖҒж•°жҚ®еҲҶеҢәгҖӮ
    • ReplicatedMergeTree: з”ЁдәҺе®һзҺ°ж•°жҚ®еүҜжң¬гҖӮ
    • SummingMergeTree: з”ЁдәҺе®һж—¶иҒҡеҗҲжұӮе’ҢгҖӮ
    • AggregatingMergeTree: з”ЁдәҺе®һж—¶иҒҡеҗҲд»»ж„Ҹзҡ„иҒҡеҗҲеҮҪж•°гҖӮ
    • CollapsingMergeTree: з”ЁдәҺеӨ„зҗҶйҮҚеӨҚж•°жҚ®пјҢдҫӢеҰӮж—¶й—ҙеәҸеҲ—ж•°жҚ®дёӯзҡ„еҸ–ж¶Ҳж“ҚдҪңгҖӮ
    • VersionedCollapsingMergeTree: еёҰжңүзүҲжң¬жҺ§еҲ¶зҡ„ CollapsingMergeTreeгҖӮ
    • GraphiteMergeTree: з”ЁдәҺеӯҳеӮЁе’ҢжҹҘиҜў Graphite ж јејҸзҡ„ж•°жҚ®гҖӮ
  • Log зі»еҲ—еј•ж“Һ:
    • TinyLog: з®ҖеҚ•еј•ж“ҺпјҢз”ЁдәҺеҝ«йҖҹеҶҷе…Ҙе°ҸиЎЁпјҢж•°жҚ®ж–Үд»¶жҢүеҲ—зӢ¬з«ӢеӯҳеӮЁгҖӮ
    • Log: ж”ҜжҢҒе№¶еҸ‘иҜ»еҸ–пјҢж•°жҚ®ж–Үд»¶жҢүеҲ—еӯҳеӮЁгҖӮ
    • StripeLog: ж”ҜжҢҒе№¶еҸ‘иҜ»еҸ–пјҢжүҖжңүеҲ—ж•°жҚ®еӯҳеӮЁеңЁеҗҢдёҖдёӘж–Үд»¶дёӯгҖӮ
  • йӣҶжҲҗеӨ–йғЁж•°жҚ®жәҗзҡ„еј•ж“Һ:
    • HDFS: е…Ғи®ёзӣҙжҺҘд»Һ HDFS иҜ»еҸ–ж•°жҚ®гҖӮ
    • MySQL: е…Ғи®ёд»Һ MySQL ж•°жҚ®еә“иҜ»еҸ–ж•°жҚ®гҖӮ
    • JDBC: е…Ғи®ёд»Һд»»дҪ•ж”ҜжҢҒ JDBC зҡ„ж•°жҚ®еә“иҜ»еҸ–ж•°жҚ®гҖӮ
    • File: е…Ғи®ёиҜ»еҸ–ж–Үд»¶зі»з»ҹдёҠзҡ„ж•°жҚ®гҖӮ
  • зү№ж®Ҡз”ЁйҖ”зҡ„еј•ж“Һ:
    • Distributed: з”ЁдәҺеҲҶеёғејҸжҹҘиҜўпјҢдёҚеӯҳеӮЁж•°жҚ®гҖӮ
    • Memory: е°ҶжүҖжңүж•°жҚ®дҝқеӯҳеңЁ RAM дёӯгҖӮ
    • Set: з”ЁдәҺеӯҳеӮЁе”ҜдёҖеҖјгҖӮ
    • Buffer: з”ЁдәҺеңЁзү№е®ҡжқЎд»¶дёӢе°Ҷж•°жҚ®ејӮжӯҘеҶҷе…Ҙзӣ®ж ҮиЎЁгҖӮ
    • MaterializedView: зү©еҢ–и§ҶеӣҫпјҢж №жҚ® SELECT жҹҘиҜўзҡ„з»“жһңеҲӣе»әгҖӮ
  • е…¶д»–еј•ж“Һ:
    • Join: з”ЁдәҺеңЁиҝҗиЎҢж—¶жү§иЎҢ JOIN ж“ҚдҪңгҖӮ
    • URL: е…Ғи®ёд»Һ URL иҜ»еҸ–ж•°жҚ®гҖӮ
    • Kafka: з”ЁдәҺд»Һ Kafka иҜ»еҸ–ж•°жҚ®гҖӮ
    • Redis: з”ЁдәҺд»Һ Redis иҜ»еҸ–ж•°жҚ®гҖӮ

д»ҘдёҠиҝҷдәӣеј•ж“ҺпјҢжҢүз…§еҜ№еӨҡеҲҶеҢәгҖҒеӨҡеүҜжң¬зҡ„ж”ҜжҢҒиҝӣиЎҢеҲҶзұ»пјҡ

  • ж”ҜжҢҒеӨҡеҲҶеҢәзҡ„еј•ж“Һ:
    • MergeTree: ж”ҜжҢҒж•°жҚ®еҲҶеҢәгҖӮ
    • SummingMergeTree: з»§жүҝиҮӘ MergeTreeпјҢж”ҜжҢҒеҲҶеҢәгҖӮ
    • AggregatingMergeTree: з»§жүҝиҮӘ MergeTreeпјҢж”ҜжҢҒеҲҶеҢәгҖӮ
    • CollapsingMergeTree: з»§жүҝиҮӘ MergeTreeпјҢж”ҜжҢҒеҲҶеҢәгҖӮ
    • VersionedCollapsingMergeTree: з»§жүҝиҮӘ CollapsingMergeTreeпјҢж”ҜжҢҒеҲҶеҢәгҖӮ
    • GraphiteMergeTree: з»§жүҝиҮӘ MergeTreeпјҢж”ҜжҢҒеҲҶеҢәгҖӮ
  • ж”ҜжҢҒеӨҡеүҜжң¬зҡ„еј•ж“Һ:
    • ReplicatedMergeTree: дё“дёәеӨҚеҲ¶иЎЁи®ҫи®ЎпјҢж”ҜжҢҒеӨҡеүҜжң¬гҖӮ
  • ж—ўж”ҜжҢҒеӨҡеҲҶеҢәпјҢд№ҹж”ҜжҢҒеӨҡеүҜжң¬зҡ„еј•ж“Һ:
    • ReplicatedMergeTree: иҝҷжҳҜе”ҜдёҖдёҖдёӘж—ўж”ҜжҢҒеӨҡеҲҶеҢәд№ҹж”ҜжҢҒеӨҡеүҜжң¬зҡ„еј•ж“ҺгҖӮе®ғз»“еҗҲдәҶ MergeTree зі»еҲ—еј•ж“Һзҡ„еҲҶеҢәзү№жҖ§е’ҢеӨҚеҲ¶иЎЁеј•ж“Һзҡ„еӨҡеүҜжң¬зү№жҖ§гҖӮ

иҜ·жіЁж„ҸпјҢиҷҪз„¶ ReplicatedMergeTree ж”ҜжҢҒеӨҡеҲҶеҢәе’ҢеӨҡеүҜжң¬пјҢдҪҶжҳҜе®ғйҖҡеёёз”ЁдәҺзү№е®ҡзҡ„йӣҶзҫӨи®ҫзҪ®дёӯпјҢйңҖиҰҒдёҺ ZooKeeper/ClickHouseKeeper дёҖиө·дҪҝз”Ёд»Ҙз®ЎзҗҶеүҜжң¬д№Ӣй—ҙзҡ„дёҖиҮҙжҖ§гҖӮ

е…¶д»–еј•ж“ҺпјҢеҰӮ TinyLog, Log, StripeLog зӯүпјҢиҷҪз„¶еҸҜиғҪж”ҜжҢҒеҲҶеҢәпјҢдҪҶе®ғд»¬йҖҡеёёз”ЁдәҺзү№е®ҡзҡ„з”ЁдҫӢпјҢеҰӮж—Ҙеҝ—еӯҳеӮЁпјҢе№¶дёҚж”ҜжҢҒеӨҚеҲ¶иЎЁзҡ„зү№жҖ§гҖӮиҖҢ Distributed, Memory, Set, Buffer зӯүеј•ж“Һдё»иҰҒз”ЁдәҺзү№е®ҡзҡ„жҹҘиҜўжҲ–ж•°жҚ®з®ЎзҗҶж“ҚдҪңпјҢе№¶дёҚзӣҙжҺҘж”ҜжҢҒеӨҡеҲҶеҢәжҲ–еӨҡеүҜжң¬зҡ„зү№жҖ§гҖӮеңЁйҖүжӢ©иЎЁеј•ж“Һж—¶пјҢйңҖиҰҒж №жҚ®е…·дҪ“зҡ„дёҡеҠЎйңҖжұӮе’ҢйӣҶзҫӨжһ¶жһ„жқҘзЎ®е®ҡжңҖйҖӮеҗҲзҡ„еј•ж“ҺгҖӮ
